# MVVM_DataBind_Retrofit2_RxJava_Demo
  
  使用了DataBind实现MVVM的demo。<br>
  应用到了retrofit2、rxjava、rxandroid、Android L的Transition动画（sdkVersion>=21）。<br>
  接口使用了聚合数据的微信经典文章。<br>
  Transition动画与webview的配合不好（与其他类似textview配合良好），请大家提出意见或修改之。
